# Uber Analytics Dashboard

## Project Overview
Analyzed 100K+ Uber booking records to uncover revenue trends, booking patterns, cancellation behavior, customer satisfaction, and operational KPIs.

## Tools Used
- Python
- Pandas
- NumPy
- Plotly
- Streamlit
- GitHub

## Key Insights
- Identified peak booking hours
- Analyzed vehicle-wise revenue
- Studied cancellation patterns
- Evaluated payment methods
- Compared weekday performance

## Dataset Size
103,024 bookings

## Future Enhancements
- Apache Kafka
- Apache Spark Streaming
- PostgreSQL
- Real-time Dashboard
## Dataset

The original dataset contains 103,024 Uber booking records.
